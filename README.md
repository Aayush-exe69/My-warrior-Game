# Assassin Hunter 🎮

Assassin Hunter is a simple 2D action platformer made with :contentReference[oaicite:0]{index=0}.  
This is my beginner game project where the player explores the map, avoids dangers, collects coins, and tries to survive.

## Features
- 2D platformer movement
- Jump and run mechanics
- Enemy and trap system
- Coin collection
- Simple level design
- Pixel-style graphics

## Controls
- **A / Left Arrow** → Move Left
- **D / Right Arrow** → Move Right
- **Space** → Jump

## Built With
- :contentReference[oaicite:1]{index=1} Engine
- GDScript
- Linux for development
